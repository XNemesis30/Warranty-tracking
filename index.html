App Setup

Connect your Google Sheet as the database

Step 1 – Create Google Apps Script

1

Open a Google Sheet, then go to Extensions → Apps Script

2

Copy the contents of Code.gs into the script editor

3

Click Deploy → New deployment, choose Web app, set Execute as: Me, Access: Anyone

4

Copy the Web App URL and paste it below

Step 2 – Paste Your Web App URL

🌐 Customer Form Setup

1

Save your Google Apps Script URL in Step 2 above first

2

Click 📋 Copy Customer Form URL below – it includes your API key

3

Generate a QR code from the copied URL (use qr-code-generator.com)

4

Print the QR code and give it to customers – works on any browser/device

5

Customer scans → fills form → data auto-saves to your Google Sheet

GitHub Pages Hosting

1

Create a GitHub repository (e.g. ez-gadgets-rma)

2

Upload index.html and Code.gs to the repo

3

Go to Settings → Pages, set source to main branch

4

Your app will be live at https://yourusername.github.io/ez-gadgets-rma
